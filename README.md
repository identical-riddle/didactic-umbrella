# didactic-umbrella
A mini-game in C.


> **_PREREQUISITE:_** `make` and `gcc`

#### To play the game
- Open your bash terminal and run the following commands.

```bash
username@OS:~/didactic-umbrella$ make NGG
username@OS:~/didactic-umbrella$ ./NGG
```

 - Make sure that you are in the project folder in the terminal.
